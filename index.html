How to Run Multiple Peers

To run multiple instances of this application and see peers connect:

  1. Open a terminal and navigate to the peartree directory
  2. Run npx pear run . to start the first instance
  3. Open another terminal and run npx pear run . again to start another instance
  4. Each new instance will connect to the network and appear as a pear on the tree
  5. Hover over any pear to see information about that peer
